¿Quién inventó el sistema de control de versión Git y por qué?
Git fué creado por Linus Torvalds, él buscaba un sistema distribuido que pudiera usar en forma semejante a BitKeeper,
 pero ninguno de los sistemas bajo software libre disponibles cumplía con sus requerimientos, especialmente en cuanto a desempeño.

¿A quién pertenece actualmente Github y por qué?
GitHub fué adquirida por Microsoft, para tener mayor contacto con los desarrolladores de software libre y la integracion con otras plataformas y servicios.
