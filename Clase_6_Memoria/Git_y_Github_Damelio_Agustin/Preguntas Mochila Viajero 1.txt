a. ¿Quién inventó el sistema de control de versión Git y por qué?
Git es un sistema de control de versiones distribuido que fue creado por Linus Torvalds en 2005 para ser utilizado en el desarrollo del kernel de Linux.

b. ¿A quién pertenece actualmente Github y por qué?
Actualmente Github pertenece a la empresa Microsoft, la cual lo compró por 7500 millones de dólares. El motivo de esta compra
ha adoptado una estrategia nueva, mucho más abierta al mundo del software libre y a la integración con otras plataformas y servicios.
