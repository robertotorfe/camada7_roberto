A- ¿Quién inventó el sistema de control de versión Git y por qué?

Hace ya 10 años Linus Torvalds (2005) creaba la primera revisión de git, un sistema de control de versiones (diseñado inicialmente para el desarrollo de Linux) que se ha convertido en este tiempo en el más importante y utilizado del mundo.

B_ ¿A quién pertenece actualmente Github y por qué?

El 4 de junio de 2018 Microsoft compró GitHub por la cantidad de 7500 millones de dólares.GitHub, que cuenta con más de 28 millones de usuarios, pasó a formar parte del negocio de Inteligencia en la Nube de Microsoft
